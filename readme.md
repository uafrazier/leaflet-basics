# Leaflet.js Basics

Beginner Leaflet.js tutorial written as a supplement for a [MaptimeTO](http://maptime.io/toronto/) workshop.

Provides a walk through of a series of example maps with each one building off the previous.

## General Outline:

- set up HTML document
- create a basic web page with a map
- modify the map for usability
- add WMS layer
- add default marker
- add custom marker
- add default popup
- customize popup

View online at http://joshuafrazier.info/leaflet-basics
